## Ping Test

# Installation

```bash
 npm install
```

### Usage

```bash
 npm start -- [host] [threshold] [delaySpikeThreshold]
```

Default values:

- `host`: `google.com`
- `threshold`: `50` (in milliseconds)
- `delaySpikeThreshold`: `100` (in milliseconds)

### Example

```bash
 npm start -- google.com 50 100
```
